### About Me || Professional Self Assessment

My name is Eboni Royal and I have been attending Southern New Hampshire University since 2023, pursuing my bachelor’s degree in computer science. I will complete the program in May 2026. Since a child I have always been interested in technology. I enlisted in the United States Army in 2015 where I was a radio operator-maintainer, after transitioning out of the service in 2019, I became an IT Specialist where I learned how to manage, configure, and troubleshoot various types of software and hardware. I decided that I wanted to go in depth with computers and learn how to develop.  I aspire to be a software developer.  
Before joining the program, the only prior experience I had with developing was a basic static website I created using HTML and CSS, however my knowledge for the core principles of programming was limited.  Throughout the program, I was able to deepen that knowledge in which I was able to gain technical and critical skills. Each course helped me build a strong foundation for software development and engineering. I learned various languages such as Python, C++, Java, and SQL. I gained knowledge about the Software Development Life Cycle and its methodologies with one of my favorites being Agile development due to its flexible approach. My communicative and collaborative skills have improved by documenting system design, creating pseudocode, UML diagrams and placing myself in each role of a future potential team member.  I developed secure coding practices learning how to handle errors, debugging, identifying vulnerabilities and mitigating security risks. 

Python was the first programming language that I learned, creating a simple text-based game teaching me the basic concepts of programming, which introduced me to my first true program ever created.  I gained proficiency with working with data structures such as arrays, linked list, hash tables, and trees along with algorithmic problem solving. I learned how to build and optimize database technologies such as MongoDB, MySQL, and SQLite. 
I gained a foundation in mobile development and architecture, learning how to build and design functional Android applications with Java. With this exposure, I worked with responsive UI/UX design, implementing SQLite database, various data structures, and setting up and working with logcat and an emulator for testing. 

I mastered Full-Stack development by implementing the JavaScript based framework, MEAN stack, learning frontend framework working with Angular and backend framework working with Express allowing me to build a dynamic web application with supporting database using MongoDb. 

Each enhancement demonstrates my ability to write effective code that not only repairs a broken application but improves its functionality and security by combining techniques showcasing my technical growth.  Together, each course I have completed, principles that I have learned, and projects I have developed have resulted in a well-balanced developer with a confident mindset and skills. Overall, I developed key qualities that I will continue to apply as I advance in my career.
