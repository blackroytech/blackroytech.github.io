
### Enhancement Two: Algorithms and Data Structures

For the **second enhancement**, I chose to convert my Array list to a HashMap data structure to handle my inventory items. I chose HashMap to demonstrate my understanding of algorithmic efficiency and data organization. Originally, a linear search had to be performed to iterate through my entire list of items, this could lead to slower response time as the warehouse inventory grew.  With HashMap I was able to use key-value pairs which reduced time complexity for searching and looking up my inventory items from O(n) to O(1).   The unique key for my application is the product’s SKU, which prevents collisions and duplicate entries, unlike with Array list which can have multiple items with the same SKU.  With this conversion, the application becomes more scalable and fit for a warehouse establishment.
