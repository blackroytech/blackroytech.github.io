### Enhancement Three: Databases

For the **third enhancement**, I replaced my original SQLite database with a Room Persistence Library to demonstrate my ability to create a new database, integrating it with my new data structure and initializing it operational. With Room, I was able to use Data Access Objects and annotations to create the database and call the CRUD operations. 
I integrated my HashMap data structure with the Room Database to optimize my data retrieval speed. The database can integrate better with android platforms than SQL which improves the application overall and reduces boiler plate.  Unlike SQLite, I was able to implement live tracking for data changes in the database, preventing runtime crashes
improving the overall stability of the application.
