### Enhancement Three: Databases

For the **third enhancement**, I replaced my original SQLite database with a Room Persistence Library to demonstrate my ability to create a new database, integrating it with my new data structure and initializing it operational. With Room, I was able to use annotations in a data access object to create the database and call the CRUD operations. 
I integrated my HashMap data structure with the Room Database which allowed me to optimize my data retrieval speed. The database can integrate better with android platforms than SQL which improves the application overall and reduces boiler plate.  Unlike SQLite, I was able to live track data changes in the database, preventing runtime crashes
improving the overall stability of the application.
